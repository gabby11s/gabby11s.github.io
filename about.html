~Who are you?

I do have to keep this vague for privacy reasons. I am an independent developer who made this game mostly for an assignment, but now has turned into a passion project. The goal for this game is simply to entertain, so I appreciate anyone who checks out my game. Many hours went into the development of the game, so thank you for being here!

For privacy reasons, no. I will make an effort to update the game every so often, but I'm not making any promises. I'm not even sure if I'm renew the RPG Maker MZ membership, so the game may remain incomplete. Perhaps there may be another application or way to finish the game if that does happen. Whatever happens, this webstie will be updated with the latest version of the game as updates roll out.
